## ✅ 테스트 결과 요약

### 1. 인증 (Authentication) [100% PASS]
*   회원가입/로그인/토큰 갱신/로그아웃/비공개 접근 차단 등 7개 시나리오 통과.
*   **특이사항**: 중복 아이디/이메일 검증 로직 정상 확인.

### 2. 매거진 (Magazine) [90% PASS]
*   **성공**: 목록 조회, 상세 조회, 수정, 공개 설정(공유 링크), 삭제, 검색, 좋아요 토글.
*   **보류**: AI 매거진 만들기 (AI 서버 타임아웃 발생).
*   **버그 수정 완료**: 
    - 키워드 검색 시 파라미터 바인딩 방식 개선 (`M-8`).
    - 제목/소개 빈 값 수정 시 400 에러 처리 강화.

### 3. 사용자 및 팔로우 (User & Follow) [100% PASS]
*   내 프로필 조회/수정, 타인 팔로우/언팔로우, 목록 조회(팔로워/팔로잉) 통과.
*   **특이사항**: 자기 자신 팔로우 방지 및 중복 팔로우 방지 로직 검증 완료.

## 🐛 발견된 주요 이슈 (버그 리포트)

### [ISSUE-01] AI 매거진 생성 타임아웃
*   **증상**: AI 생성 요청 시 90초 초과 후 500 또는 Timeout 에러.
*   **원인**: Python 서버(Mine-AI)의 API Key 누락(1차) 및 로컬 환경 CPU 이미지 생성 처리 부하로 인한 지연(2차).
*   **조치**: API Key 설정 완료. 성능 이슈는 향후 GPU 환경 검토 또는 비동기 처리 도입 예정.

### [ISSUE-02] 검색 쿼리 호환성 문제
*   **증상**: `LIKE` 검색 시 키워드 위치에 따라 결과 미도출.
*   **해결**: `LIKE CONCAT('%', :keyword, '%')` 방식으로 쿼리 정규화.

### [ISSUE-03] Swagger 정렬 필드 오류
*   **증상**: 기본값 `"string"` 전송 시 400 Bad Request 발생.
*   **해결**: 실제 엔티티 필드명(`createdAt`, `id`) 사용 가이드 작성.
